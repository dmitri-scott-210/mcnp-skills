# Temperature field (8th column):
#   - In units of kT (MeV)
#   - Convert to Kelvin: T(K) = kT(MeV) / 8.617333E-11
#   - Common values:
#       2.5301E-08 MeV → 293.6 K  (room temperature)
#       5.1697E-08 MeV → 600 K    (moderate temperature)
#       7.7534E-08 MeV → 900 K    (typical PWR/BWR)
#       1.0337E-07 MeV → 1200 K   (high temperature)
#       2.1543E-07 MeV → 2500 K   (very high temperature)
#
# ZAID format: ZZZAAA.nnX
#   ZZZ = Atomic number (1-118)
#   AAA = Mass number (0 for natural element mix)
#   nn  = Library identifier (80, 81, 82, etc.)
#   X   = Library type:
#         c = Continuous energy neutron
#         d = Discrete energy neutron
#         t = Thermal scattering S(α,β)
#         p = Photoatomic (photon)
#         e = Photoelectron (electron)
#         y = Dosimetry
#         h = Proton
#         u = Photonuclear
#
# Library versions:
#   .80c = ENDF/B-VIII.0 (latest, released 2018)
#   .70c = ENDF/B-VII.0 (2006)
#   .71c = ENDF/B-VII.1 (2011)
#   .66c = ENDF/B-VI.8 (2001)
#
# Section markers in xsdir:
#   directory     - Neutron continuous energy data
#   thermal       - Thermal scattering S(α,β)
#   photoatomic   - Photon interaction data
#   photoelectron - Electron interaction data
#   dosimetry     - Dosimetry cross sections
