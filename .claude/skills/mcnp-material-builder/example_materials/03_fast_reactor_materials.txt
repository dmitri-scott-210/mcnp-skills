c ========================================================================
c NOTES AND USAGE
c ========================================================================
c
c 1. Metal Fuel Advantages:
c    - Higher heavy metal density (15.8 vs 10.5 g/cm3 for oxide)
c    - Better thermal conductivity (runs cooler)
c    - Accommodates high burnup
c    - Self-shielding effects important
c
c 2. Temperature Conditions:
c    - Metal fuel: 600-900 K (much cooler than oxide)
c    - Cladding: 700-800 K
c    - Sodium coolant: 650 K inlet, 750 K outlet
c    - Blanket: 500-650 K (peripheral region)
c
c 3. Sodium Properties:
c    - Liquid range: 371-1156 K
c    - Density at 700 K: 0.8536 g/cm3
c    - Density at 750 K: 0.8408 g/cm3
c    - Adjust cell card density for temperature
c
c 4. HT-9 vs. Other Cladding:
c    - HT-9: Ferritic steel, better swelling resistance
c    - Alternative: D9 austenitic steel
c    - Temperature limit: ~923 K for HT-9
c
c 5. Plutonium Isotopics:
c    - Shown: Weapons-grade Pu composition
c    - Reactor-grade: More Pu-240, Pu-241, Pu-242
c    - Adjust isotopic vector for specific fuel cycle
c
c 6. Breeding Blanket:
c    - U-238 blanket captures neutrons → Pu-239
c    - Breeding ratio >1.0 possible in fast spectrum
c    - Model blanket regions for accurate breeding
c
c 7. B4C Control:
c    - Natural boron: 19.9% B-10, 80.1% B-11
c    - B-10 is primary absorber
c    - Enriched B-10 may be used for stronger worth
c
c 8. Fast Spectrum Considerations:
c    - NO S(alpha,beta) needed (high energy neutrons)
c    - Doppler feedback important (fuel temperature coefficient)
c    - Sodium void worth is key safety parameter
c

c METALLIC FUEL NOTES:
c ========================================================================
c
c 1. U-Zr Alloy Properties:
c    - High density: 15.8 g/cm3 (compact core)
c    - Thermal conductivity: ~10× better than oxide
c    - Low melting point: ~1400 K (design constraint)
c    - Zr content: 6-10% typical
c
c 2. U-Pu-Zr Ternary Alloy:
c    - Used in fast breeder reactors
c    - Pu content: 15-25% typical
c    - Breeding ratio >1.0 achievable
c    - Requires reprocessing capability
c
c 3. Weight Fractions (negative values):
c    - Example: -0.1778 means 17.78 wt%
c    - Must sum to -1.0
c    - Use for alloys and mixtures by weight
c
c 4. Comparison: Metallic vs Oxide Fuel:
c    - Metallic: Higher density, better conductivity, lower melting
c    - Oxide: Higher melting, more experience, simpler fabrication
c    - Metallic: Breeding, Pu production
c    - Oxide: Power reactors, established technology
c
c 5. For more fuel compositions:
c    - See fuel_compositions_reference.md for complete coverage
c    - Includes UO2, MOX, UCO, metallic, HALEU, research fuels
