c ========================================================================
c NOTES AND USAGE
c ========================================================================
c
c 1. HALEU Definition and Regulations:
c    - HALEU: 5% < enrichment < 20%
c    - Above: Typical LWR maximum (5%)
c    - Below: HEU threshold (20%, safeguards/security)
c    - Regulatory status: Category II nuclear material
c
c 2. Applications by Enrichment:
c    - 5-10%: Extended burnup LWRs, some SMRs
c    - 10-15%: Microreactors, compact cores
c    - 15-20%: Long-life cores (>10 years), HTGRs
c    - 19.75%: Maximum HALEU (commonly used for TRISO)
c
c 3. Temperature Considerations:
c    - Microreactors: 800-1000 K typical
c    - HTGR (TRISO): 900-1200 K
c    - SMR: 800-1100 K similar to conventional LWR
c
c 4. Enrichment vs Core Size:
c    - Higher enrichment → smaller core
c    - HALEU enables compact, transportable designs
c    - Microreactors: 1-20 MWe, very compact
c
c 5. U-234 and U-236 Inclusion:
c    - U-234: ~0.8% of U-235 content (natural ratio)
c    - U-236: Trace (<0.02%) from reprocessing
c    - Include for high-fidelity calculations
c    - Can omit for scoping studies
c
c 6. Fuel Forms:
c    - UO2: Most common, established technology
c    - Metallic (U-Zr): Fast spectrum microreactors
c    - UCO: HTGR TRISO kernels
c    - UN, U3Si2: Alternative high-density forms
c
c 7. Burnup Characteristics:
c    - Higher enrichment → longer cycle length
c    - Example: 19.75% can achieve >10 years
c    - Reduced refueling, simplified operations
c    - Increased Pu production (proliferation concern)
c
c 8. HALEU Supply:
c    - Limited commercial availability (as of 2025)
c    - DOE HALEU program developing supply chain
c    - Downblending from HEU stockpiles
c    - New enrichment capacity needed
c
c 9. Safety Implications:
c    - Higher enrichment → more excess reactivity
c    - Requires burnable poisons (Gd, Er)
c    - Negative reactivity feedbacks essential
c    - Passive safety features in microreactors
c
c 10. For Complete Coverage:
c     - See fuel_compositions_reference.md for all fuel types
c     - Includes LWR (2-5%), HALEU (5-20%), MOX, metallic, etc.
c     - See ZAID_selection_guide.md for library selection
c     - See burnup_tracking_guide.md for depletion isotopes
c
