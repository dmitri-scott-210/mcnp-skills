c ========================================================================
c NOTES AND USAGE
c ========================================================================
c
c 1. TRISO Particle Structure:
c    - Kernel: UO2 fuel (M1), ~500 μm diameter
c    - Buffer: Porous PyC (M2), ~100 μm thick
c    - IPyC: Inner PyC layer, ~40 μm (use M2 at 1.9 g/cm3)
c    - SiC: Pressure vessel (M3), ~35 μm
c    - OPyC: Outer PyC layer, ~40 μm (use M2 at 1.9 g/cm3)
c
c 2. Temperature Profile:
c    - Fuel kernel: 1200-1400 K (centerline can reach 1600 K)
c    - Coatings: Follow fuel temperature
c    - Graphite: 700-1000 K depending on location
c    - Helium: 900 K outlet (inlet ~600 K)
c
c 3. Graphite S(alpha,beta):
c    - GRPH.43t is for 600 K (closest to operating range)
c    - Available: GRPH.40t (293.6 K), GRPH.43t (600 K), GRPH.44t (800 K)
c    - Select based on local temperature
c
c 4. Enrichment Options:
c    - 19.75% shown is typical HALEU (High-Assay LEU)
c    - Can use LEU <20% or adjust for different designs
c
c 5. Helium Density:
c    - Shown: 7 MPa, 900 K conditions
c    - Density varies with pressure/temperature
c    - At 293 K, 0.1 MPa: 0.0001664 g/cm3
c    - Calculate for your specific conditions
c
c 6. Material Modifications:
c    - PyC density: Buffer 1.0, IPyC/OPyC ~1.9 g/cm3
c    - SiC stoichiometry: Always 1:1 Si:C ratio
c    - Graphite: May add impurities for real designs
c
