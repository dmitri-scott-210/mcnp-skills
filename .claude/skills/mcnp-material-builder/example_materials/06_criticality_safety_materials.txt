c NOTES AND USAGE
c ========================================================================
c
c 1. Solution Concentrations:
c    - Low: <100 g U/L or <10 g Pu/L (safe geometry)
c    - Medium: 100-400 g U/L, 10-100 g Pu/L (typical processing)
c    - High: >400 g U/L, >100 g Pu/L (requires strict controls)
c
c 2. Criticality Parameters:
c    - Keff < 0.95 for normal conditions (subcritical + margin)
c    - Keff < 0.98 for single contingency
c    - Model worst-case: optimal moderation, full density, reflection
c
c 3. Uranyl Nitrate Properties:
c    - Formula: UO2(NO3)2·xH2O in water
c    - Density increases with U concentration
c    - 300 g U/L: ρ ≈ 1.18 g/cm3
c    - 450 g U/L: ρ ≈ 1.28 g/cm3
c
c 4. Plutonium Solution:
c    - More radiotoxic than uranium
c    - Lower critical mass
c    - Weapons-grade shown (adjust isotopics for reactor-grade)
c    - Typical: Pu-239 50-70%, Pu-240 20-25%, Pu-241 5-10%
c
c 5. Borated Water for Criticality Control:
c    - Spent fuel pools: 2000-2500 ppm typical
c    - Emergency flooding: >1000 ppm minimum
c    - Natural boron: 19.9% B-10 (absorber)
c    - Check boron credit in safety analysis
c
c 6. UO2 Powder Density:
c    - Theoretical density: 10.96 g/cm3
c    - Poured powder: 6-8 g/cm3 (50-73% TD)
c    - Pressed pellet: 10.4-10.5 g/cm3 (95% TD)
c    - Use conservative (highest) density for safety analysis
c
c 7. Neutron Absorbers:
c    - Cadmium: Very high thermal capture, burns out over time
c    - Boron (B4C): Stable, used in storage racks
c    - Gadolinium: Used in some fuel designs
c
c 8. Geometry Considerations:
c    - Infinite arrays more reactive than single units
c    - Water reflection increases reactivity significantly
c    - Full flooding vs. optimal moderation (may not be same)
c
c 9. Temperature Effects:
c    - Room temperature shown (conservative for most cases)
c    - Elevated temp: Density decreases (usually less reactive)
c    - But Doppler broadening affects resonances
c
c 10. Double Contingency Principle:
c     - Require two unlikely, independent failures for criticality
c     - Typical controls: geometry, mass, moderation, absorption
c     - Never rely on single parameter alone
c
c 11. Material Modifications:
c     - Solution: Adjust U/Pu concentration for your scenario
c     - Enrichment: Critical mass depends strongly on this
c     - Boron: Scale concentration for required margin
c     - Isotopics: Use actual material composition
