# MCNP Physics Validator - Scripts

**Version:** 2.0.0

## Overview
Python scripts for validating physics settings.

## Usage

```bash
python mcnp_physics_validator.py input.inp
```

**Checks:**
- MODE card present and first
- Cross-section library availability
- Energy cutoff appropriateness
- TMP card consistency
