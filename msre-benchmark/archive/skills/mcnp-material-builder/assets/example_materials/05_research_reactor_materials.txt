c ========================================================================
c NOTES AND USAGE
c ========================================================================
c
c 1. LEU Fuel Forms:
c    - Dispersion: UO2 particles in Al matrix (plate fuel)
c    - TRIGA: U-ZrH homogeneous (rod fuel)
c    - MTR: Highly enriched historically, now converting to LEU
c
c 2. Enrichment Limit:
c    - 19.75% U-235 is maximum for Low-Enriched Uranium (LEU)
c    - Historical research reactors used HEU (93%)
c    - Global RERTR program converting HEU to LEU
c
c 3. TRIGA Features:
c    - Zirconium hydride provides prompt negative feedback
c    - Can safely pulse to high power
c    - H-ZRH.42t S(alpha,beta) critical for reactivity
c
c 4. Beryllium Reflector:
c    - (n,2n) reaction increases neutron economy
c    - Low absorption, good scattering
c    - Toxic - handle with care
c    - Temperature limit ~773 K (oxidation concerns)
c
c 5. Aluminum Advantages:
c    - Low absorption cross section
c    - Good thermal conductivity
c    - Easy to fabricate
c    - Compatible with water
c
c 6. Research Reactor Operations:
c    - Typical power: 1-10 MW thermal
c    - Fuel temperature: 350-450 K (much cooler than power reactors)
c    - Water: Room temperature, no pressurization
c    - No boron typically (control rods provide shutdown)
c
c 7. Pool Water:
c    - Pure water, demineralized
c    - NO boron (unlike PWR)
c    - Acts as moderator, coolant, and shield
c    - Depth: 6-8 m typical for shielding
c
c 8. Material Modifications:
c    - Dispersion fuel: Adjust UO2 loading (20-40% typical)
c    - TRIGA: H:Zr ratio varies (1.0 to 1.7)
c    - Reflector: Can use graphite or BeO as alternatives
c
c 9. S(alpha,beta) Tables:
c    - H-H2O.40t for water at 293.6 K
c    - H-ZRH.42t for TRIGA fuel at 373.6 K
c    - BE.40t for beryllium at 293.6 K
c    - GRPH.40t for graphite at 293.6 K
c    - Match TMP card to table temperature
