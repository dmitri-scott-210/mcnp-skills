# Example 9: Angular Sector

## Purpose
Demonstrates angular sector (wedge) geometry using planes.

## Geometry
- Quarter cylinder: 0° to 90° in XY plane
- Bounded by X=0 and Y=0 planes
- Height: 0 to 20 cm

## Applications
- Symmetric geometry (model 1/4, reflect results)
- Pie-slice detectors
- Azimuthal segmentation

## Usage
```bash
mcnp6 inp=09_sector_geometry.i
```
